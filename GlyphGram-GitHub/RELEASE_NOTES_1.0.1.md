# GlyphGram 1.0.1

Glyph lighting for Telegram round-video messages on Nothing Phone (2).

## Installation

1. Install `GlyphGram-Companion-1.0.0.apk`.
2. Open the companion and test **Turn Glyph on** / **Turn Glyph off**.
3. In AyuGram, open **Settings → exteraGram settings → Plugins**.
4. Import `GlyphGram-1.0.1.plugin`, enable it, and fully restart AyuGram.
5. Open the round-video camera and tap the new `✦` button. Green means armed.

## Compatibility

- Nothing Phone (2)
- Nothing OS 4.1 / Android 16
- AyuGram 12.5.1 (`com.radolyn.ayugram`)

The APK is signed with the same dedicated test certificate as previous
GlyphGram Companion releases. Do not uninstall the old companion before
updating, unless Android reports a signature conflict.
