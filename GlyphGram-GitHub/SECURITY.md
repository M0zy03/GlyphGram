# Security policy

Please do not publish security-sensitive reports in a public issue. Contact the
maintainer on Telegram: [@M0zy1](https://t.me/M0zy1).

GlyphGram does not request network access and does not collect or transmit user
data. The companion exposes an Android service used by the AyuGram plugin to
switch Glyph lighting on and off.

Only the latest release is supported.
