# Contributing

Bug reports and pull requests are welcome.

When reporting a bug, include:

- phone model;
- Nothing OS version;
- AyuGram/exteraGram version;
- exact reproduction steps;
- relevant AyuGram plugin log or `adb logcat` output.

Do not attach private chats, authentication data, signing keys, or modified
third-party APKs. Keep changes focused and test them on a physical Nothing
Phone (2) before opening a pull request.
